## 🎯 Présentation
Class-Time affiche en temps réel lavancement du cours en cours, le temps restant, et le prochain cours à venir. Disponible en version web (interface graphique) et en version CLI (terminal). Synchronisable à la milliseconde pour être aligné avec la sonnerie de lécole.

---

## ✨ Fonctionnalités
- ✅ Affichage du cours en cours avec barre de progression
- ✅ Prochain cours à venir
- ✅ Synchronisation à la milliseconde
- ✅ Version web et version Python CLI
- ✅ Configuration via fichier JSON (emploi du temps)

---

## 🚀 Installation

### Version web
Ouvrez simplement `index.html` dans votre navigateur.

### Version CLI
```bash
python3 script.py
```
